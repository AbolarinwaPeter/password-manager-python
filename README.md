# Password Manager (Python)

A graphical password manager built using Python and Tkinter.  
The application generates secure passwords and allows users to store and retrieve login credentials for different websites.

## Features

- Generate secure passwords
- Automatically copy generated password to clipboard
- Save login credentials with website name
- Browse stored passwords
- Simple graphical interface

## Technologies Used

- Python
- Tkinter (GUI)
- Random module
- Clipboard automation

## How It Works

1. Enter the website name.
2. Click **Generate Password** to create a secure password.
3. The password is automatically copied to your clipboard.
4. Click **Save** to store the login credentials.
5. Stored passwords can be browsed later within the application.

## How to Run

python main.py
